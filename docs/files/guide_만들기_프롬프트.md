아래 형식에 맞게 네이버 블로그 자동 작성 프로그램용 guide.md 파일을 작성해줘.

⚠️ 중요: 이 guide.md는 브랜드 정보·요금제·말투·금지사항만 담당합니다.
글의 섹션 구성·순서·개수는 프로그램 스타일탭 설정이 별도로 제어하므로
guide.md에 "다음 섹션을 포함하세요", "N개 섹션으로 구성하세요" 같은 지시는 절대 쓰지 마세요.

내 서비스 정보:
- 서비스명: [여기에 입력]
- 메인 URL: [여기에 입력]
- 고객센터 연락처 또는 방법: [여기에 입력]
- 서비스 특징 (강점, 편의성 등): [여기에 입력]
- 통신망 종류 (KT망/LGU+망/SKT망 등, 해당 없으면 생략): [여기에 입력]
- 요금제 목록 (이름, 데이터, 월 요금, 통화/문자): [여기에 입력]
- 유심 구입 방법 (구입처, 유심 이름, 가격): [여기에 입력]
- 신분증 종류 (사용 가능한 것 목록): [여기에 입력]
- 간편인증 수단 (네이버, 카카오, PASS 등, 해당 없으면 생략): [여기에 입력]
- 개통 운영시간 (신규/번호이동 각각): [여기에 입력]
- 개통 절차 단계 (순서대로, 해당 없으면 생략): [여기에 입력]
- 원하는 말투 (예: 친근한, 전문적, 간결한): [여기에 입력]
- 금지 사항 (경쟁사 이름, 잘못된 정보 등): [여기에 입력]
- 강조하고 싶은 문구 (선택): [여기에 입력]
- 독자 대상 (선택): [여기에 입력]
- 추가로 알려줄 내용 (선택): [여기에 입력]

출력 형식 규칙:
1. 아래 구조를 그대로 따라야 함
2. 요금제는 반드시 표(| 요금제명 | 데이터 | 월 기본료 | 통화/문자 | QoS |) 형식으로 작성
3. 유심 구입 안내는 | 표 형식으로 작성 (통신망, 유심명, 가격, 구입처 열)
4. 개통 운영시간도 | 표 형식으로 작성 (구분, 가능시간, 운영일 열)
5. 입력받은 정보만 사용하고 없는 정보는 해당 섹션 삭제
6. 마크다운 코드 블록(```) 없이 텍스트만 출력
7. 섹션 구성·개수·순서에 관한 지시는 절대 포함하지 말 것
