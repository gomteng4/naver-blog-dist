# 이미지 프롬프트 지침

블로그 포스팅에 사용할 이미지를 AI로 생성할 때 `image_prompts.txt` 작성 방법입니다.
이 파일의 내용을 참고하여 LLM(ChatGPT, Claude 등)에게 `image_prompts.txt` 생성을 요청할 수 있습니다.

---

## image_prompts.txt 형식

MD 파일과 같은 폴더에 저장합니다.

```
[이미지 1]
프롬프트: (영어로 작성) Flat design infographic showing...
텍스트: 한글키워드1, 한글키워드2, 수치

[이미지 2]
프롬프트: ...
텍스트: ...

[썸네일]
프롬프트: ...
텍스트: ...
```

- `[이미지 N]` 번호는 MD 파일의 `> [이미지 N]` 번호와 반드시 일치
- 본문 이미지는 **4개 고정** (썸네일 제외)
- 썸네일이 필요 없으면 `[썸네일]` 섹션 생략

---

## 프롬프트 작성 원칙

1. **영어로 작성** — 한글 포함 시 깨질 수 있음
2. **Flat design 스타일** 명시 — 깔끔한 인포그래픽 유지
3. **구성 요소 3개 이내** — 복잡하면 품질 저하
4. **반드시 포함**: `no people`, `no watermark`, `white background`
5. **텍스트 필드**: 이미지에 넣을 한글 키워드/수치 (쉼표 구분)

---

## 이미지별 프롬프트 패턴

**이미지 1 — 유심/도입부**
```
[이미지 1]
프롬프트: Flat design infographic showing a SIM card labeled "KT" and convenience store icons. White background with red and blue accents. Simple icons only, no people, no watermark.
텍스트: KT, 8,800원, 무약정
```

**이미지 2 — 요금제 비교**
```
[이미지 2]
프롬프트: Flat design comparison cards. Left card "기본1 / 300MB / 12,100원", right highlighted card "396 / 10.3GB / 39,600원" with star badge. White background, no watermark.
텍스트: 기본1, 12,100원, 396, 39,600원, 10.3GB
```

**이미지 3 — 개통 단계**
```
[이미지 3]
프롬프트: Flat design step-by-step infographic. 5 numbered steps connected by arrows. Blue accent, white background, no watermark.
텍스트: STEP 1, STEP 2, STEP 3, STEP 4, STEP 5
```

**이미지 4 — 간편인증/운영시간**
```
[이미지 4]
프롬프트: Flat design 2x3 grid of 6 colorful rounded app icon placeholders. Below: clock icon with time label. White background, no real logos, no watermark.
텍스트: 네이버, PASS, 카카오, 토스, 신한, KB, 08:00~21:50
```

## 썸네일 생성

썸네일은 `image_prompts.txt`에 `[썸네일]` 섹션으로 추가합니다.

```
[썸네일]
프롬프트: Flat design thumbnail for Korean prepaid phone guide. SIM card with "KT" label and "8,800원" price tag. Clean white background with red and blue accents. Bold Korean text overlay style. Modern infographic, no people, no watermark.
텍스트: KT, 선불폰, 8,800원, 개통 가이드
```

**썸네일 작성 팁:**
- 글 제목의 핵심 키워드를 텍스트 필드에 넣을 것
- 통신사 색상 활용: KT → 빨강/파랑, LGU+ → 보라/민트
- 가로형 비율로 구성 (블로그 썸네일 기준)

---

## LLM에게 요청하는 방법

아래처럼 지침과 함께 요청하면 됩니다.

```
다음 조건에 맞는 image_prompts.txt를 작성해 주세요.

- 글 주제: [주제 설명]
- 이미지 4개 + 썸네일 1개
- 각 이미지는 [이미지 N] 형식으로 작성
- 프롬프트는 영어, Flat design 스타일, white background, no people, no watermark
- 텍스트 필드에 해당 이미지에 들어갈 한글 키워드/수치 포함
```

> 이미지가 MD 파일의 어느 위치에 들어가는지 확인하려면 `작성가이드.md` 참조
