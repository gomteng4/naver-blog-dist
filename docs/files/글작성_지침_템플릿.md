# 블로그 글 작성 지침 템플릿

> 이 파일은 **AI 글 작성 지침(guide.md) 생성을 보조**하기 위한 참고 문서입니다.
> 본인의 예시 글 2편과 이 파일을 함께 AI에 입력하면, 브랜드에 맞는 맞춤 지침을 받을 수 있습니다.

|------|-----|

## ▶ 프로그램 MD 서식 규칙 (변경 금지)

아래는 이 프로그램이 처리하는 마크다운 규칙입니다.
**AI가 생성하는 글도 반드시 이 규칙을 따라야 합니다.**

### 볼드
- `**텍스트**` → 해당 부분만 굵게 처리
- 한 줄에 여러 개 사용 가능
- 예: `가격은 **8,800원**이며 **약정 없이** 사용 가능합니다.`

### 소제목
- `## 소제목` → 인용구 블록 + 수평선으로 삽입
- `### 소소제목` → 볼드 텍스트로 삽입
- `##### 강조 소제목` → 19pt + 볼드 + 색상 강조 (앱 설정 색상)
- `#### 대형 소제목` → 24pt + 볼드 + 색상순환 (요금제명 등에 사용)

### 인용구
- `> 텍스트` → 인용구 블록으로 삽입 + 자동 수평선
- 본문 맨 위 도입 문장에 사용 권장

### 이미지
- `> [이미지 1]` ~ `> [이미지 4]` → 해당 위치에 이미지 삽입
- 반드시 `##` 소제목 바로 아래 첫 줄에 위치

### 리스트
- `1. 항목` → 번호 리스트
- `- 항목` → 불릿 리스트

### 표
- 마크다운 표 형식(`| 항목 | 값 |`) 그대로 사용

### 구분선
- `---` (하이픈 3개 단독 줄) → 수평선 삽입
- 소제목/인용구 다음에는 자동 삽입되므로 직접 넣을 필요 없음

---

## ▶ 글 생성 체크리스트 (서식 확인용)

### 작성 중
- [ ] 본문 맨 위에 `> 인용구` 도입부
- [ ] 각 `## 소제목` 바로 아래 `> [이미지 N]`
- [ ] 이미지 태그 총 4개
- [ ] `**볼드**` 섹션당 핵심 수치/단어 1~2개

### 작성 후
- [ ] 브랜드명, 홈페이지 주소 일관 사용 확인
- [ ] 이미지 태그 번호와 이미지 파일 번호 일치 확인
