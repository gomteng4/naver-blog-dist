# 블로그 글 작성 지침 (브랜드 템플릿)

> 아래 **브랜드 정보 섹션만 수정**하면 어떤 브랜드에도 그대로 사용 가능합니다.

---

## ▶ 브랜드 정보 (여기만 수정)

| 항목 | 값 |
|------|-----|
| 브랜드명 | 이음모바일 |
| 홈페이지 주소 | 이음모바일.COM |
| 개통 사이트 안내 문구 | "이음모바일.COM에 접속해" |
| KT 유심 구입처 | GS25, CU |
| LGU+ 유심 구입처 | 이마트24, 스토리웨이, 배민B마트 앱 |
| 고객센터 안내 | 이음모바일.COM 내 채팅 문의 |

> 브랜드명과 홈페이지 주소는 글 전체에서 일관되게 사용한다.
> 다른 브랜드 이름이나 URL을 절대 혼용하지 않는다.

---

## 최우선 규칙

### 요금제 데이터
- 요금제 정보는 반드시 제공된 `요금제.md` 파일만 참조한다
- 파일에 없는 요금제명, 금액, 데이터량은 절대 사용하지 않는다

### 사실 기반 작성
- 확인되지 않은 통계, 지역 특성, 인구, 트렌드 수치는 절대 언급하지 않는다
- 수치가 필요한 경우 요금제.md에 있는 데이터만 사용한다

## 글 구조 (권장)

```
① 도입부
② 선불폰 개념 및 장점
③ 유심카드 종류 + 간편인증 안내
④ 개통 순서
⑤ 마무리
```

권장 구조이며, 제목 성격에 따라 순서와 섹션명은 자유롭게 조정 가능하다.
두괄식 원칙: 도입부 직후 핵심 답변 1~2문장 먼저 제시

## SEO 조건

| 항목 | 기준 |
|------|------|
| 글자 수 | 1,500~2,000자 |
| 키워드 밀도 | 제목 + 본문 합산 5~6회 |
| 목차 | H2 계층 필수 |
| 이미지 | 본문 4개 고정 (썸네일 제외) |
| 제목 구조 | 질문형 또는 방법형 + 핵심 키워드 |

## 모바일 가독성 규칙

- 한 문장 30자 이내 (최대 40자)
- 한 단락 2~3문장 이내
- 3가지 이상 나열은 반드시 리스트 사용

---

## MD 꾸밈 규칙

### 볼드
- 섹션당 핵심 키워드·수치 1~2개만 `**볼드**` 처리
- 예: "가격은 **8,800원**이며", "**약정 없이** 사용 가능"

### 인용구
- 이미지 태그와 별도로, 섹션 핵심 문장을 `>` 로 강조 가능
- 위치: 이미지 태그 아래, 섹션 본문 중간 또는 마지막 줄

## 제목 패턴 (A→B→C 순환)

| 유형 | 패턴 | 예시 |
|------|------|------|
| A | 방법/절차형 | 선불폰 개통 어떻게 하나요? 유심 선택부터 요금제까지 |
| B | 비교/선택형 | 선불폰 KT망 vs LGU+망, 어떤 게 더 나을까요 |
| C | 추천/정보형 | 선불유심, 약정 없이 쓰기 좋은 이유 세 가지 |

## 도입부 패턴 (매 글마다 교체)

**A — 공감형**
```
약정 없이 휴대폰을 사용하고 싶은데 어디서 시작해야 할지 막막하셨나요?
선불폰은 생각보다 훨씬 간단하게 시작할 수 있습니다.
오늘은 [핵심 주제]를 처음부터 끝까지 안내해 드릴게요.
```

**B — 핵심 먼저형**
```
선불폰 개통은 유심 구매 → 간편인증 → 요금제 선택 세 단계로 끝납니다.
복잡한 서류도, 긴 약정도 필요 없습니다.
오늘 바로 시작할 수 있도록 순서대로 안내해 드릴게요.
```

**C — 사실형**
```
약정 없는 선불폰은 비용 절감과 자유로운 번호 이동이 가장 큰 장점입니다.
[브랜드명] 선불폰의 핵심 내용만 짚어드릴게요.
```

## 개통 순서 방식 (매 글마다 교체)

**A — 숫자 단계형**
```
STEP 1. 유심 구입
STEP 2. [홈페이지 주소] 접속
STEP 3. 간편인증으로 본인 인증
STEP 4. 요금제 선택 → 개인정보 입력
STEP 5. 선불 요금 결제 → 개통 완료
```

**B — Q&A형**
```
Q. 유심은 어디서 사나요?
A. KT망은 GS25·CU, LGU+망은 이마트24·스토리웨이·배민B마트 앱. 가격 8,800원.

Q. 인증은 어떻게 하나요?
A. 네이버, PASS, 카카오, 토스, 신한, KB 중 하나로 본인 인증.

Q. 마지막 단계는?
A. 요금제 선택 후 결제하면 개통 완료.
```

**C — 흐름 서술형**
```
편의점에서 유심을 구입한 뒤 [홈페이지 주소]에 접속합니다.
간편인증으로 본인 인증을 마친 뒤 요금제를 선택하고 결제하면 개통 완료입니다.
```

## 마무리 패턴 (매 글마다 교체)

**A — 요약 강조형**
```
유심 구입 → 본인인증 → 요금제 선택 → 결제 순서만 기억하면 처음이어도 어렵지 않습니다.
```

**B — 비교 유도형**
```
약정 없이 자유롭게 쓰고 싶다면 선불폰이 가장 합리적인 선택입니다.
```

**C — 행동 유도형**
```
오늘 10분만 투자하면 개통까지 완료할 수 있으니 지금 바로 도전해 보세요.
```

## 글 생성 체크리스트

### 작성 전
- [ ] 직전 글 조합 패턴 확인 (중복 방지)
- [ ] 요금제 데이터 파일 확인
- [ ] 이번 글 조합 선택

### 작성 중
- [ ] 본문 맨 위 `> 핵심 문장` 인용구
- [ ] 도입부 2~3문장
- [ ] 목차 포함 (## 소제목과 이름 일치)
- [ ] 각 ## 소제목 바로 아래 `> [이미지 N]`
- [ ] 이미지 태그 총 4개
- [ ] 요금제명 `####` 작성
- [ ] 요금제 내용 번호 리스트
- [ ] 개통 운영시간 포함
- [ ] 간편인증 6종 안내 포함
- [ ] 글자 수 1,500~2,000자

### 작성 후
- [ ] [브랜드명], [홈페이지 주소] 일관 사용 확인
- [ ] 이미지 태그와 이미지 파일 번호 일치 확인
