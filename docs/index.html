프로그램 다운로드

zip 파일을 받아 압축을 풀면 바로 실행할 수 있습니다.
별도의 설치 과정이 필요하지 않습니다.

⬇  최신 버전 다운로드 약 40 MB
초기 사용 요령
  1. 1
    zip 다운로드 → 바탕화면에 압축 해제 위 버튼으로 zip 파일을 저장 후, 바탕화면(또는 원하는 위치)에 압축을 풀면
    naver-blog/ 폴더가 생성됩니다.
  2. 2
    폴더 구조 확인 압축 해제 후 생성되는 구조입니다.
    naver-blog/
      ├── naver-blog.exe  ← 실행 파일
      ├── 콘텐츠/           ← 글 원고 폴더
      │     └── {아이디폴더}/
      │          └── {날짜폴더}/
      │               ├── 글제목.md
      │               └── image_prompts.txt
      └── settings.json   ← 설정 자동저장
  3. 3
    exe 실행 → 라이선스 키 입력 naver-blog.exe 더블클릭 후 실행합니다.
    Windows 보호 팝업 → 추가 정보 → 실행 클릭.
    발급받은 라이선스 키를 입력하면 프로그램이 시작됩니다.
  4. 4
    좌측 [추가] 버튼 → 네이버 아이디 등록 좌측 패널에서 [+ 추가] 버튼을 눌러 사용할 네이버 계정(아이디/비밀번호)을 등록합니다.
    여러 계정을 등록해두고 선택하여 사용할 수 있습니다.
  5. 5
    아이디에 맞는 폴더 생성 콘텐츠/ 안에 등록한 아이디와 동일한 이름의 폴더를 만듭니다.
    예: 아이디가 myblog123 이면 → 콘텐츠/myblog123/
    그 안에 날짜 폴더(2026-04-12(1))를 만들고 MD 파일을 넣으면 준비 완료.
  6. 6
    업데이트 새 버전이 있으면 실행 시 자동으로 알림이 표시됩니다.
    업데이트 버튼 클릭 한 번으로 최신 버전으로 교체됩니다.
AI 글 생성 사용 요령
  1. 1
    필요한 파일 2종 준비 날짜 폴더 안에 아래 파일을 넣습니다.
    {날짜폴더}/
      ├── 글제목.md  ← 글 본문 원고
      └── image_prompts.txt  ← 이미지 프롬프트 (선택)
    파일 작성 가이드는 아래 자료 다운로드 섹션에서 받으세요.
    LLM(ChatGPT, Claude 등)에 글작성 지침과 함께 요청하면 MD 파일을 자동 생성할 수 있습니다.
  2. 2
    MD 파일 구조 MD 파일은 반드시 프론트매터(---로 감싼 제목)로 시작해야 합니다.
    ---
    title: "OO동 선불폰 개통 방법"
    ---

    > 도입부 핵심 문장

    ## 소제목

    > [이미지 1]

    본문 내용...
  3. 3
    프로그램에서 계정 선택 → 실행 좌측에서 계정을 선택하고, 우측에서 포스팅할 폴더를 선택합니다.
    이미지 생성 방식(Grid/개별)을 선택한 뒤 [포스팅 시작] 버튼을 누르면
    이미지 생성 → 블로그 글쓰기까지 자동으로 진행됩니다.
  4. 4
    완료 확인 포스팅이 완료되면 날짜 폴더 안에 완료.txt가 생성됩니다.
    이미 완료된 폴더는 다음 실행 시 자동으로 건너뜁니다.
이미지 생성 요령

이미지는 Gemini API를 통해 자동 생성됩니다. API 키는 앱 설정에서 입력합니다.

이미지 생성 방식
Grid 방식
2×2 그리드 → 자동 분할
이미지 4개를 하나의 그리드로 생성 후 자동 분할. API 1회 호출로 4개 생성 — 빠르고 경제적.
개별 방식
이미지 1장씩 개별 생성
각 이미지를 독립적으로 생성. API 4회 호출 — 이미지별 맞춤 스타일 적용 가능.
썸네일 생성
블로그 대표 썸네일 자동 생성
image_prompts.txt에 [썸네일] 섹션을 추가하면 대표 썸네일을 별도로 생성합니다.
본문 이미지(이미지 1~4)와 독립적으로 동작합니다.
[이미지 1]
프롬프트: Flat design infographic...
텍스트: KT, 8,800원, 무약정

[이미지 2]
프롬프트: ...

[썸네일]
프롬프트: Flat design thumbnail for Korean prepaid...
텍스트: 선불폰, 개통 가이드
image_prompts.txt 규칙
  1. 규칙
    번호는 MD 파일의 이미지 태그와 일치 MD에 > [이미지 2] 가 있으면 txt에도 [이미지 2] 섹션 필요.
  2. 규칙
    프롬프트는 영어, 텍스트 필드는 한글 프롬프트: 영어로 작성 (Flat design, white background, no watermark 포함)
    텍스트: 이미지에 넣을 한글 키워드/수치 (쉼표 구분)
  3. 팁
    상세 가이드는 자료 다운로드 참조 아래 이미지 프롬프트 지침 파일을 다운받아 LLM에게 프롬프트 생성을 요청할 수 있습니다.
