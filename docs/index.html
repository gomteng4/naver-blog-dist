Windows 전용 · 라이선스 필요

네이버 블로그
자동 포스팅 프로그램

콘텐츠 파일 작성 후 버튼 한 번으로
네이버 블로그에 자동 포스팅합니다.

확인 중… 최신 버전

프로그램 다운로드

zip 파일을 받아 압축을 풀면 바로 실행할 수 있습니다.
별도의 설치 과정이 필요하지 않습니다.

⬇  최신 버전 다운로드 약 40 MB
초기 사용 요령
  1. 1
    zip 다운로드 → 바탕화면에 압축 해제 위 버튼으로 zip 파일을 저장 후 바탕화면(또는 원하는 위치)에 압축 해제.
    naver-blog/ 폴더가 생성됩니다.
  2. 2
    폴더 구조
    naver-blog/
      ├── naver-blog.exe    실행 파일
      └── 콘텐츠/            글 원고 폴더
            └── {아이디}/       아이디와 동일한 이름
                  └── {날짜폴더}/ YYYY-MM-DD(순번)
                        └── 글제목.md
  3. 3
    exe 실행 → 라이선스 키 입력 naver-blog.exe 더블클릭 후 실행.
    Windows 보호 팝업 → 추가 정보 → 실행 클릭.
    발급받은 라이선스 키를 입력하면 프로그램이 시작됩니다.
  4. 4
    좌측 [+ 추가] → 네이버 계정 등록 사용할 네이버 아이디/비밀번호를 등록합니다.
    여러 계정을 등록해두고 선택하여 사용할 수 있습니다.
    ⚠ 블로그 주소(ID)가 로그인 아이디와 다른 경우
    네이버 계정 아이디(로그인 ID)와 블로그 주소(blog.naver.com/블로그ID)가 다른 경우, 계정 등록 다이얼로그의 블로그 ID 입력 필드에 실제 블로그 주소 ID를 따로 입력하세요.
    비워두면 로그인 후 자동 감지를 시도합니다.
  5. 5
    아이디와 동일한 폴더 생성 콘텐츠/ 안에 등록한 아이디와 동일한 이름의 폴더를 만듭니다.
    예: 아이디 myblog123 → 콘텐츠/myblog123/
  6. 6
    날짜 폴더에 MD 파일 배치 → 포스팅 시작 2026-04-12(1)/ 형식의 날짜 폴더를 만들고 MD 파일을 넣으면 준비 완료.
    프로그램에서 계정·폴더 선택 후 [포스팅 시작]을 누르면 자동 실행됩니다.
AI 글 생성 사용 요령

제목 목록과 작성 지침을 준비해두면, Gemini AI가 제목을 한 개씩 꺼내
블로그 글(MD)과 이미지 프롬프트(image_prompts.txt)를 동시에 자동 생성합니다.

준비 파일 — 아이디 폴더 안에 prompt/ 폴더 생성
콘텐츠/{아이디}/
  └── prompt/                  직접 만드세요
        ├── guide.md             블로그 글 작성 지침 ← 필수
        ├── titles.txt            제목 목록 (한 줄에 하나) ← 필수
        ├── image_guide.md       이미지 프롬프트 추가 지침 (선택)
        └── used_titles.json     자동 생성 — 사용된 제목 기록
titles.txt 예시
가경동 선불폰 개통 어떻게 하나요?
북가좌동 선불폰 KT망 vs LGU+망
명동 선불유심 약정 없이 쓰기 좋은 이유
신정동 선불폰 셀프개통 방법
... (한 줄에 제목 하나씩)
실행 흐름
  1. 1
    titles.txt에서 미사용 제목 1개 선택 순서대로 또는 무작위로 꺼냅니다. 모두 사용하면 알림을 표시합니다.
  2. 2
    guide.md 지침에 맞게 Gemini가 블로그 글 작성 제목 + guide.md 지침 → Gemini API → 마크다운 형식 블로그 본문 생성.
  3. 3
    생성된 본문 기반으로 image_prompts.txt 자동 생성 블로그 본문의 이미지 배치([이미지 1]~[이미지 4])를 분석해
    각 섹션에 맞는 이미지 프롬프트 + 썸네일 프롬프트를 자동으로 작성합니다.
  4. 4
    날짜 폴더 자동 생성 → 두 파일 저장 YYYYMMDD/ 폴더가 자동 생성되고
    {제목}.md + image_prompts.txt 가 함께 저장됩니다.
    같은 날 여러 개 생성 시 YYYYMMDD_01/, YYYYMMDD_02/ 형식으로 번호가 붙습니다.
  5. 5
    이미지 생성 → 포스팅 실행 프로그램에서 생성된 날짜 폴더를 선택하고 [포스팅 시작]을 누르면
    이미지 자동 생성 → 블로그 자동 업로드까지 한 번에 진행됩니다.
guide.md 준비 — 아래 자료 다운로드에서 글작성_지침_템플릿.md를 받아 브랜드 정보 섹션만 수정하면 바로 guide.md로 사용할 수 있습니다.
이미지 생성 요령

날짜 폴더에 image_prompts.txt를 넣으면 Gemini API로 이미지를 자동 생성합니다.
API 키는 앱 설정에서 입력합니다.

생성 방식 3가지
Grid 방식
2×2 그리드 → 자동 분할
이미지 4개를 하나의 그리드로 한 번에 생성 후 자동 분할. API 1회 호출 — 빠르고 경제적.
Gemini 생성 (1장)
이미지 1
이미지 2
이미지 3
이미지 4
→
자동 분할 (4장)
image_1.png
image_2.png
image_3.png
image_4.png
grid.png 1장 생성 → image_1~4.png 자동 저장
개별 방식
이미지 1장씩 개별 생성
각 이미지를 독립적으로 생성. API 4회 호출 — 이미지별 맞춤 스타일 가능.
썸네일 생성
블로그 대표 썸네일 자동 생성
image_prompts.txt에 [썸네일] 섹션을 추가하면 본문 이미지와 별개로 대표 썸네일(thumbnail.png)을 생성합니다.
image_prompts.txt 작성 형식
[이미지 1]
프롬프트: Flat design infographic showing a SIM card...
텍스트: KT, 8,800원, 무약정

[이미지 2]
프롬프트: Flat design comparison cards...
텍스트: 396, 39,600원, 10.3GB

[썸네일]
프롬프트: Flat design thumbnail for Korean prepaid...
텍스트: 선불폰, 개통 가이드
  1. 규칙
    [이미지 N] 번호는 MD 파일과 일치 MD에 > [이미지 2]가 있으면 txt에도 [이미지 2] 섹션이 있어야 합니다.
  2. 규칙
    프롬프트는 영어 / 텍스트 필드는 한글 프롬프트: 영어 (Flat design, white background, no watermark 포함 권장)
    텍스트: 이미지에 넣을 한글 키워드/수치 (쉼표 구분)
상세 가이드 — 아래 자료 다운로드에서 이미지프롬프트_지침.md를 받아
ChatGPT나 Claude에게 그대로 전달하면 프롬프트를 대신 작성해 줍니다.
